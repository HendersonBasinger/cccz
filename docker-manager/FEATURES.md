# Docker管理面板功能说明

## 🎯 访问信息

- **管理面板**: http://localhost:3000/admin
- **默认账号**: admin
- **默认密码**: admin123

## 📦 功能模块

### 1. 📊 仪表盘
- 查看系统概览
- 总用户数/活跃用户统计
- 快捷操作入口

### 2. 👥 用户管理
- **添加用户**: 设置名称、到期时间
- **编辑用户**: 修改用户信息和到期时间
- **批量操作**: 支持批量启用/禁用/删除
- **订阅链接**: 一键复制用户订阅地址

### 3. 📦 套餐管理 ⭐核心功能⭐
- **创建套餐**: 名称、时长(天)、价格、描述
- **编辑套餐**: 修改套餐信息
- **上下架**: 控制套餐是否对用户可见
- **删除套餐**: 移除不需要的套餐

**示例套餐**:
- 月度套餐: 30天 ¥19.9
- 季度套餐: 90天 ¥49.9

### 4. 💳 订单管理 ⭐核心功能⭐
- **订单筛选**: 全部/待审核/已通过/已拒绝
- **单个审核**: 通过/拒绝单个订单
- **批量审核**: 批量通过/拒绝待审核订单
- **自动延期**: 通过订单后自动延长用户到期时间

**订单流程**:
1. 用户下单选择套餐
2. 管理员在订单管理中审核
3. 通过后用户自动延期对应天数

### 5. 📢 公告管理
- **创建公告**: 标题、内容(最多500字)
- **编辑公告**: 修改现有公告
- **启用/禁用**: 控制公告是否显示给用户
- **删除公告**: 移除过期公告

**使用场景**:
- 系统维护通知
- 优惠活动公告
- 重要事项提醒

### 6. 💰 支付渠道管理
- **添加渠道**: 名称、类型(人工审核/BEpusdt)、配置信息
- **启用/禁用**: 控制支付渠道状态
- **删除渠道**: 移除不使用的支付方式

**支持类型**:
- 人工审核: 支付宝、微信、银行转账等
- BEpusdt接口: 自动化支付接口

### 7. 🎫 邀请码管理
- **生成邀请码**: 设置可用次数和赠送天数
- **使用跟踪**: 查看已用/总次数
- **启用/禁用**: 控制邀请码有效性
- **删除邀请码**: 移除过期邀请码

**示例邀请码**:
- WELCOME2024: 可用10次，赠送7天试用
- VIP888: 可用5次，赠送30天试用

### 8. ⚙️ 配置管理
- **节点订阅地址**: 设置订阅链接域名
- **系统设置**: 其他全局配置

## 💡 常用操作

### 审核订单流程
1. 登录管理面板
2. 点击左侧菜单 "💳 订单管理"
3. 默认显示"待审核"订单
4. 选择订单，点击"通过"或使用批量操作
5. 通过后用户自动延期

### 添加新套餐
1. 点击左侧菜单 "📦 套餐管理"
2. 填写套餐信息（名称、时长、价格、描述）
3. 点击"添加套餐"
4. 新套餐自动上架，可在列表中管理

### 发布公告
1. 点击左侧菜单 "📢 公告管理"
2. 点击右上角 "+ 添加公告"
3. 填写标题和内容
4. 勾选"启用此公告"
5. 点击"保存"

### 生成邀请码
1. 点击左侧菜单 "🎫 邀请码"
2. 设置可使用次数
3. 设置赠送试用天数
4. 填写备注(可选)
5. 点击"生成邀请码"

## 🔧 高级功能

### 批量操作
- **用户批量管理**: 勾选多个用户，批量启用/禁用/删除
- **订单批量审核**: 勾选多个待审核订单，批量通过/拒绝

- 容器重启后数据不丢失

## 🚀 快速开始

### 第一次使用
```bash
# 1. 访问管理面板
打开浏览器访问: http://localhost:3000/admin

# 2. 登录
账号: admin
密码: admin123

# 3. 查看各个模块
依次点击左侧菜单查看功能
```

### Docker命令
```bash
# 查看容器日志
docker logs vles-manager

# 重启容器
docker restart vles-manager

# 停止容器
docker stop vles-manager

# 启动容器
docker start vles-manager

# 查看容器状态
docker ps | grep vles-manager
```

### 数据备份
```bash
# 备份数据库
docker cp vles-manager:/app/data/vles.db ./backup-$(date +%Y%m%d).db

# 恢复数据库
docker cp ./backup-20240111.db vles-manager:/app/data/vles.db
docker restart vles-manager
```

## ⚠️ 注意事项

1. **首次登录**: 建议修改默认管理员密码
2. **数据备份**: 定期备份数据库文件
3. **端口冲突**: 确保3000端口未被占用
4. **订单审核**: 通过订单前请确认用户信息
5. **套餐价格**: 设置合理的套餐价格

## 🔐 安全建议

1. 修改默认管理员密码
2. 使用强密码策略
3. 定期备份数据
4. 限制管理面板访问IP
5. 使用HTTPS访问(生产环境)
